# FurniHub - Pakistan's Premier Furniture Marketplace

A modern, feature-rich furniture marketplace connecting customers with verified furniture shops across Pakistan. Built with React and enhanced with Tailwind CSS for a premium user experience.

## 🚀 Features

### 🎨 Enhanced User Interface
- **Premium Design System** - Modern UI with Tailwind CSS
- **Responsive Layout** - Optimized for all screen sizes
- **Smooth Animations** - Micro-interactions and transitions
- **Professional Branding** - Consistent design language throughout

### 🏪 Shop Features
- **Verified Shop Badges** - Trust indicators for authentic shops
- **Enhanced Shop Cards** - Premium design with hover effects
- **Quick Actions** - Wishlist and quick view functionality
- **Shop Statistics** - Ratings, reviews, and contact info
- **Featured Shops Carousel** - Showcase top-rated shops

### 🛋️ Product Features
- **Advanced Product Cards** - Rich product information display
- **Contact Shop Integration** - Direct messaging with shop owners
- **Wishlist Functionality** - Save favorite items
- **Quick View** - Preview products without navigation
- **Category Navigation** - Visual category browsing with icons
- **Stock Status** - Real-time availability indicators

### 🔍 Search & Navigation
- **Smart Search Bar** - Search shops and products
- **City-Based Filtering** - Find local furniture shops
- **Tab Navigation** - Seamless switching between shops and products
- **Auto-Scroll** - Smooth scrolling to content sections
- **URL-Based Navigation** - Shareable links to specific sections

### 👤 User Experience
- **Mobile-First Design** - Perfect on all devices
- **Smooth Scrolling** - Enhanced navigation experience
- **Loading States** - Professional loading indicators
- **Error Handling** - Graceful error messages

## 🛠️ Technology Stack

### Frontend
- **React 18** - Modern React with hooks
- **React Router** - Client-side routing
- **Tailwind CSS** - Utility-first CSS framework
- **Lucide React** - Modern icon library
- **Axios** - HTTP client for API calls

### Backend Integration
- **RESTful API** - Connected to Node.js/Express backend
- **Authentication** - JWT-based user authentication
- **Real-time Data** - Dynamic shop and product information

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd furnihub-frontend
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm start
   ```

4. **Open browser**
   Navigate to `http://localhost:3000`
